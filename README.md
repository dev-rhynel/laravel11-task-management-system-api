# Installation
## Install Make, See Makefile

or

## Direct install docker
- Install docker/docker-compose

## Run Engine
- Type "make start"
- or "docker compose up -d"

## Test
- Type "make test" 
- Or "docker exec $(APP_CONTAINER) php artisan test"
